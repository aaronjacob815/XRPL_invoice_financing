# XRPL_invoice_financing

Project for the Ripple track of the Finnovator hackathon @ Imperial, June '26.

This is the backend of a blockchain 'invoice financing' scheme, linking freelancers, companies, and lenders together using XRPL.
Abstracted description: someone has an invoice for later but needs money now, so we ask lenders on the blockchain to provide money to that client immediately, and the company that owes the invoice pays the lender directly. Every stakeholder benefits in some way.

Written by Aaron Jacob, Sebastian Kember, Astha Nihar, and Naveen Goswami.
